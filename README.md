# VoxelwiseEncoding

Voxel-wise encoding model using [VOneResNet50](https://github.com/dicarlolab/vonenet) as feature extractor and a 3D CNN as response model.

For NSD subject 8 only.

Subject 1: (81, 104, 83)
Subject 2: (82, 106, 84)
Subject 3: (81, 106, 82)
Subject 4: (85, 99, 80)
Subject 5: (79, 97, 78)
Subject 6: (85, 113, 83)
Subject 7: (78, 95, 81)
Subject 8: (80, 103, 78)
